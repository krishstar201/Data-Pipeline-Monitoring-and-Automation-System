# Stream Monitoring Automation Tool

## Overview
This project automates monitoring of data pipelines and stream workflows using Python and Selenium.

It logs into systems, navigates dashboards, and validates the presence of active streams across multiple environments.

## Features
- Automated login with 2FA handling
- Multi-tenant environment support
- Stream and orchestration flow validation
- Error detection (503/404)
- Logging and monitoring automation

## Tech Stack
- Python
- Selenium
- Requests

## Use Case
Used for monitoring data pipelines, validating system health, and automating routine operational checks.

Note: This project uses Selenium and requires ChromeDriver setup and Outlook configuration for OTP automation.
